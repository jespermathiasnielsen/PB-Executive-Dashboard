# Private Banking & Investments — Executive Dashboard

Executive performance dashboard for Danske Bank's Private Banking & Investments division, built around the Forward'28 strategy targets. Tracks AUM, net new money, profitability KPIs, advisory penetration, and open leadership actions across all four Nordic markets.

Built in Python and Streamlit using the Danske Bank brand palette.

## Dashboard tabs

| Tab | Content |
|---|---|
| **Forward'28 Overview** | ROE, cost/income ratio, total income, CET1, AUM, and NNM — each with a progress bar tracking distance from baseline to 2028 target, RAG status, and quarterly trend charts |
| **AUM & Net New Money** | Stacked AUM bar chart by market, quarterly NNM waterfall, client segment pie (AUM share) and bar (client count) |
| **Nordic Markets** | Per-market KPI cards (AUM, NNM delta vs prior quarter), AUM trend lines, NNM waterfall by market |
| **Commercial Momentum** | Radar chart across advisory penetration, insurance penetration, and lending growth; quarterly trend lines per market; grouped lending growth bars |
| **Action Log** | Filterable action table with owner, due date, status pills, and overdue flagging; summary counters by status |

## Forward'28 targets wired in

All targets are sourced from Danske Bank's publicly disclosed Forward'28 strategy:

| KPI | Baseline (Q1 2026) | Target 2028 |
|---|---|---|
| Return on Equity | 13.0% | 14.5% |
| Cost / Income Ratio | 46.0% | ≤43.0% |
| Total Income | DKK 57bn | DKK 63bn |
| CET1 Capital Ratio | 15.5% | ~16.0% |
| Assets Under Management | DKK 1,010bn | ~DKK 1,200bn |
| Net New Money (annual) | — | DKK 35bn |

## Installation

```bash
git clone https://github.com/jespermathiasnielsen/PB-Executive-Dashboard.git
cd PB-Executive-Dashboard
pip install -r requirements.txt
streamlit run src/app.py
```

Open http://localhost:8501.

## Data

All figures are synthetic and generated by `src/data.py`. They are anchored to publicly disclosed Forward'28 targets and modelled to reflect plausible quarterly trajectories. No confidential Danske Bank data is used or implied.

## Design

Colours follow the Danske Bank brand palette (primary navy #003F63, Pantone 2955 C). Typography uses Inter. The layout is designed for presentation on a 1280px-wide screen — suitable for executive briefings and leadership forums.
